# Pro Mock Interview Simulator

A lightweight, privacy-focused, and customizable interview simulation tool built with Python. This tool helps candidates practice under timed pressure with a real-time camera preview to monitor their body language and expressions.

## âœ¨ Features

- **Dual-Phase Timer**: Automatically transitions from a 10-second preparation phase to a 90-second response phase.
- **Live Video Feed**: Integrated camera preview using OpenCV to simulate a real online interview environment.
- **Local & Private**: No data is uploaded to the cloud. Your video stream and question bank stay entirely on your local machine.
- **Easy Customization**: Change questions, timer durations, and UI colors directly in the global configuration block.
